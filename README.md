# 📊 Google Play Store Exploratory Data Analysis (EDA)

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on the Google Play Store dataset using Python. The objective is to clean the data, analyze app categories, ratings, reviews, installs, pricing, and identify patterns that provide meaningful business insights.

---

## 🎯 Objectives

- Understand the structure of the dataset
- Clean and preprocess the data
- Handle missing values
- Analyze app categories
- Explore app ratings and reviews
- Study installs and pricing
- Generate visual insights using charts

---

## 🛠️ Technologies Used

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## 📂 Dataset

**Source:** Google Play Store Dataset

The dataset contains information about Android applications available on the Google Play Store, including:

- App Name
- Category
- Rating
- Reviews
- Size
- Installs
- Type
- Price
- Content Rating
- Genres

## ▶️ How to Run

Clone the repository:

```bash
git clone https://github.com/payalsinghal633-ui/Google-PlayStore-EDA.git
```

Install dependencies:

```bash
pip install pandas numpy matplotlib seaborn
```

Open the notebook:

```bash
jupyter notebook project1.ipynb
```

Run all cells.
